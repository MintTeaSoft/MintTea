About This Portal

This page explains how the ENGCE301 Lab Submission Portal is derived from your SRS, User Stories and Use Cases.

System Overview

ระบบสั่งอาหารในมหาวิทยาลัยเป็น มือถือแอป ที่มีวัตถุประสงค์เพื่อให้นักศึกษาสามารถสั่งอาหาร จองโต๊ะ และรีวิวร้านอาหารภายในมหาวิทยาลัย

From SRS to Screens

  • FR-001: User Registration → ผู้ใช้สามารถลงทะเบียนเข้าใช้ระบบด้วยรหัสนักศึกษา รหัสผ่าน อีเมล และข้อมูลส่วนตัว
  • FR-002: User Login → ผู้ใช้เข้าสู่ระบบด้วย Student ID และ Password
  • FR-003: Update User Profile → ผู้ใช้สามารถแก้ไขข้อมูลส่วนตัว เช่น เบอร์โทร อีเมล รูปโปรไฟล์ และข้อมูลติดต่อ
  • FR-004: Browse Restaurants & Menu → ผู้ใช้สามารถดูรายการร้านอาหารและเมนูพร้อมรายละเอียด
  • FR-005: Place Food Order → ผู้ใช้ทำการสั่งอาหารในแอป
  • FR-006: Track Order Status → ผู้ใช้ติดตามสถานะคำสั่งซื้อแบบเรียลไทม์
  • FR-007: Place Food Order → ระบบตรวจสอบการชำระเงิน
  • FR-008: Submit Review → ผู้ใช้จองโต๊ะกับร้าน
  • FR-009: Submit Review → ผู้ใช้ให้คะแนนและรีวิวร้านอาหารที่เคยสั่ง
  • FR-010: Restaurant Login → ร้านค้าเข้าสู่ระบบด้วย Restaurant ID ,Password
  • FR-011: Restaurant Menu Management → ร้านค้าจัดการเมนู เช่น เพิ่ม ลบ หรือแก้ไขเมนู
  • FR-012: Restaurant Registration → Admin จะสมัครบัญชีสำหรับการจัดการร้านค้าให้
  • FR-013: Admin Dashboard → Admin ดูภาพรวมของสถิตการสั่งอาหาร ,สถิติการจองโต๊ะ และรีวิวของร้านอาหาร

Mapping: User Stories → UI Elements

Use this section to explicitly map each user story to parts of the UI.

Example Mapping

  • US-01 – การเข้าสู่ระบบด้วยบัญชีมหาวิทยาลัย
    → Implemented as the Login Button and SSO Login Option in the login page.
  • US-02 – การสั่งอาหารและการจองโต๊ะ
    → Implemented as Order Button and Booking Button with tabs for separate sections in the menu.
  • US-03 – ประวัติการจองโต๊ะและสั่งอาหาร
    → Represented by History Tab, Order History Cards, and Booking History Cards in the user profile.
  • US-04 – ระบบการชำระเงิน
    → Implemented as Payment Options, Proceed Button, and Confirmation Messages in the payment screen.
  • US-05 – ระบบให้คะแนนร้านอาหาร
    → Represented by Rating Stars and Review Text Box in the restaurant details page.
  • US-06 – การสั่งซื้ออาหาร
    → Implemented as Food Menu, Add to Cart Button, and Quick Order Button in the food ordering screen.
  • US-07 – Admin Dashboard
    → Represented by Dashboard Sidebar, Charts/Graphs, and Data Table in the admin panel.
